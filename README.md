# PokeAPI exercise

A small excercise of 3 questions using [PokeAPI](https://pokeapi.co).
There is no error handling, as the impossibility to get an answer should be handled by the code calling these functions.


### Answers

+ Question 1: 9
+ Question 2: 258
+ Question 3: [1300, 195]

### Requirements

+ python 3.8.10
+ requests 2.27.1
